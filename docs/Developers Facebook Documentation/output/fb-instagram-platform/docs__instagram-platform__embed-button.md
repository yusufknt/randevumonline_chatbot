# Embed Button

You can get a public Instagram photo, video, Reel, or Guide post’s embed HTML code and use it to embed the post in other websites. Embedded posts automatically resize to fit their container, so they will look great on both desktop and mobile websites.

![](https://scontent.fyei5-1.fna.fbcdn.net/v/t39.2365-6/89824380_234684177669792_3559805517834485760_n.png?_nc_cat=105&ccb=1-7&_nc_sid=e280be&_nc_ohc=Fipt8hnPoTgQ7kNvwEwz0Ns&_nc_oc=Adp8yyPIYsl_uAnDQbe64kqH9bQjI08y36cFZ9j1LdlWrVwVyNL_y3uAlVM6bGOXorg&_nc_zt=14&_nc_ht=scontent.fyei5-1.fna&_nc_gid=2aBOjYwgvXdnPJG9pzdvMA&_nc_ss=7b289&oh=00_Af4cpHdDPkMDftRjgxNA1F-LVfe2NX68RgncP4IjmbMxGg&oe=6A1BD83F)

People own their Instagram content, so each embedded post gives proper attribution by displaying the post owner’s username and linking to the post owner’s profile.

To get a post’s embed HTML code using its **Embed** button, view the post in a desktop or mobile web browser, click the **[...]** menu...

![](https://scontent.fyei5-1.fna.fbcdn.net/v/t39.2365-6/89738865_133859414730000_8450155087510110208_n.png?_nc_cat=102&ccb=1-7&_nc_sid=e280be&_nc_ohc=n-4U4BIxoRoQ7kNvwHEuWly&_nc_oc=Adot6SSbKdDuzQk-hj4Mm13NSx_BAa82rMQg6J5Ma9JxcpfpSEKSteMPlgE8YrgRu74&_nc_zt=14&_nc_ht=scontent.fyei5-1.fna&_nc_gid=2aBOjYwgvXdnPJG9pzdvMA&_nc_ss=7b289&oh=00_Af50Pgry5CgPxm6L5GYzT_DIWvUrgLCYSu1zhmSR5IUcEg&oe=6A1BE4A7)

... then click the **Embed** button:

![](https://scontent.fyei5-1.fna.fbcdn.net/v/t39.2365-6/89722283_616218892556786_5027045768683323392_n.png?_nc_cat=109&ccb=1-7&_nc_sid=e280be&_nc_ohc=8JKqQFuGJTgQ7kNvwEK43zt&_nc_oc=AdrZ0-aBIZx0Fy1z_RJk8xzwBpKKkK-Npu5rHSsWd-0aCVZ3DpN_V6whML1GCGKt9BQ&_nc_zt=14&_nc_ht=scontent.fyei5-1.fna&_nc_gid=2aBOjYwgvXdnPJG9pzdvMA&_nc_ss=7b289&oh=00_Af5Jv4ItTyUEgnG-nGbuwda1s_6iSr6tbUbJ15RoltH2Ng&oe=6A1BE7DC)

This will reveal the post’s embed HTML code, which you can copy and paste in the website where you want the post to appear.

![](https://scontent.fyei5-1.fna.fbcdn.net/v/t39.2365-6/89731764_695269624611829_7513410501070553088_n.png?_nc_cat=101&ccb=1-7&_nc_sid=e280be&_nc_ohc=FTIJNLisBowQ7kNvwFal03m&_nc_oc=AdqR_LFLhRwdbcHGM7ZT9UPgUxL2thGUvvcagISqc2V3Tp9u-PUKp0HspuYPcP8u_-c&_nc_zt=14&_nc_ht=scontent.fyei5-1.fna&_nc_gid=2aBOjYwgvXdnPJG9pzdvMA&_nc_ss=7b289&oh=00_Af6pf8uk1rqkeHbitXrfgmteP4YZrUwMLquG9MGWYuJieQ&oe=6A1BD61E)

By default the post will include its caption, but you can omit the caption by unchecking the **Include caption** checkbox.
