# Data Handling Questions

Starting on or after June 12th, 2023, if your app requires advanced level access to permissions or features, you may need to complete data handling questions. See this [blog post](https://developers.facebook.com/blog/post/2023/04/05/data-handling-questions/) for more information.

Beginning January 2024, developers with apps that have been published live with a use case or advanced access to any [permission](https://developers.facebook.com/docs/permissions/reference) or [feature](https://developers.facebook.com/docs/features-reference) will be required to complete data handling questions as part of their annual [Data Use Checkup](https://developers.facebook.com/docs/development/maintaining-data-access/data-use-checkup/).

**See [Responsible Platform Initiatives](https://developers.facebook.com/docs/resp-plat-initiatives/data-handling-questions) for the latest Data Handling Questions content.**
