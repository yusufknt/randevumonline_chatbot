# Instant Games Zero Permission Access

The **Instant Games Zero Permission Access** feature allows your app to share user data including Instant Games context ID, and player ID, user profile, location and gameplay data. The allowed usage of this feature is to integrate user profiles in Instant Games on Meta’s network without gaining direct access to the user’s data.

## Allowed Usage

***Allowed Usage***

- Integrate user profiles in Instant Games on Meta’s network

## Additional Details

- This permission or feature requires successful completion of the App Review process before your app can access live data. [Learn More.](https://developers.facebook.com/docs/app-review)
- This permission or feature is only available with business verification. You may also need to sign additional contracts before your app can access data. [Learn More Here](https://developers.facebook.com/docs/development/release/business-verification)
