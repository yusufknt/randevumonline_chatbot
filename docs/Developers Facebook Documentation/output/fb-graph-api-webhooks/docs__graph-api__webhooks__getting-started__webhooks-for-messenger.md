# Webhooks for Messenger

Webhooks for [Messenger](https://developers.facebook.com/docs/messenger-platform) allows you to receive real-time notifications when a variety of interactions or events happen, including when a person sends a message. Webhooks for Messenger works a little differently than other Webhooks so please use the [Webhooks for Messenger docs](https://developers.facebook.com/docs/messenger-platform/webhooks) when setting up this type of Webhook.
