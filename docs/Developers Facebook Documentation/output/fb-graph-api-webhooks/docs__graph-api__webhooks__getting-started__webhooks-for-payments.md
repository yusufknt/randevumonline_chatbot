# Webhooks for Payments

Webhooks for [Payments](https://developers.facebook.com/docs/games_payments) allows you to receive real-time notifications whenever a payment status has changed. Webhooks for Payments works a little differently than other Webhooks so please use the [Webhooks for Payments docs](https://developers.facebook.com/docs/games_payments/webhooks) when setting up this type of Webhook.
